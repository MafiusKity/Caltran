# Caltran
A command line calculator with real and complex numbers
## Download
```
git clone https://github.com/MafiusKity/Caltran
cd Caltran
chmod +x caltran
```
## Usage
```
#Run
./caltran -mode "expression"
```

The " " are important for functions with ( ) like sin(), sqrt(), cos(), etc.

'-mode' are -h for help, -r for real expressions, -c for complex expressions in input or output in format "(real.decimal,complex.decimal)"; atention of extra ( ) in these cases.
